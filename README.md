# Healthcare Utilization Dashboard

## Overview
This project analyzes healthcare utilization patterns using Tableau, focusing on emergency room (ER) visits and their relationship to patient-level healthcare costs.

## Objective
The goal of this project is to understand how ER usage impacts healthcare costs and identify high-utilization patients that drive overall spending.

## Tools Used
- Tableau
- Excel

## Dashboard Features
- KPI metric showing total ER visits
- Scatter plot analyzing ER visits vs healthcare cost
- Trend line to identify correlation between utilization and cost

## Key Insight
Patients with higher ER visit frequency tend to have higher healthcare costs, with a small group of high-utilization patients contributing disproportionately to total spending.

## Files Included
- healthcare_dashboard.twbx (Tableau dashboard)
- healthcare_data_cleaned.xlsx (cleaned dataset)

## How to Use
Download the `.twbx` file and open it in Tableau to interact with the dashboard.

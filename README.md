# healthcare-utilization-dashboard
This project analyzes healthcare utilization patterns using Tableau.

The dashboard focuses on emergency room (ER) visits and their relationship to patient-level healthcare costs. It includes KPI metrics and a scatter plot with a trend line to identify patterns and insights.

Key insight: Patients with higher ER usage tend to generate higher healthcare costs, with a small group of high-utilization patients driving a disproportionate share of total costs.

Tools used: Tableau, Excel
